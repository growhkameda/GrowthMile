# デザインテーマトークン（dev-diagrams 同梱・自己完結用）

本スキル単体を他プロジェクトへ移譲しても動作するよう、テーマ定義を同梱する。
プロジェクトに `slide-design-themes` スキルが存在する場合はそちらを正（SSOT）とし、本ファイルは追従させる。

## 共通

- レイアウト 16:9（10 x 5.625 in）／ 余白 0.5in 以上
- フォント: 日本語 `Yu Gothic`、コード・数値 `Courier New`
- アクセントバー・見出し下線・端ストライプ禁止／未確定要素は点線 + ラベル

## tech-dark（技術文書・開発者向け）

| トークン | 値 | 用途 |
| --- | --- | --- |
| BG | `0F172A` | 背景 |
| ZONE | `13203A` | コンテナ・ゾーン |
| CARD / CARD2 | `1E293B` / `24334D` | ノード・パネル |
| CODE | `0B1120` | コードボックス |
| TXT / MUT | `F1F5F9` / `94A3B8` | 本文 / 補足 |
| TEAL | `2DD4BF`（ラベル淡色 `7DD3C8`） | アクセント・矢印 |
| AMBER | `FBBF24` | 警告・PK・人間ゲート |
| LINE | `334155` | 罫線 |

## enterprise-light（手順書・顧客提出向け）

| トークン | 値 | 用途 |
| --- | --- | --- |
| PAPER | `FFFFFF` | 内容ページ背景 |
| NAVY / NAVYBG / NAVYP | `1F3864` / `1B3155` / `2A4A7F` | 見出し・表ヘッダー / 表紙背景 / パネル |
| INK / MUT | `333F52` / `6B7280` | 本文 / 補足 |
| LINE / TINT | `D5DBE4` / `EEF2F8` | 罫線 / ゼブラ・チップ |
| CODEBG | `F4F6F8` | コードボックス |
| WARN | bg `FFF4E5` fg `B45309` | 注意 |
| DANGER | bg `FCEBEA` fg `C0392B` | 禁止 |

- enterprise は表紙・最終頁のみ紺背景のサンドイッチ構成、表は紺ヘッダー + ゼブラ行
