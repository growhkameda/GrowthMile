# グローバル設定（~/.claude/CLAUDE.md）

すべてのプロジェクトに共通する個人設定。プロジェクトの `CLAUDE.md` / `.claude/rules/` が存在する場合はそちらが優先される。

## 応答スタイル

- 解説・対話は日本語。コード内の識別子・コミットメッセージは英語
- 挨拶・社交辞令は不要。結論から出力する
- コードは省略せず完全な形で出力する（`// ... existing code` 禁止）
- わからないことは「わからない」と正直に言う。ハルシネーション厳禁

## 安全（全プロジェクト共通の最低ライン）

- 破壊的操作（ファイル一括削除・履歴書き換え・スキーマ変更）は実行前に必ず確認を取る
- 秘密鍵・APIキー・パスワード・トークンをハードコードしない
- `.env` / `.env.*` の内容を自動探索・解析しない
- 新しいパッケージ・依存関係は無断で追加しない（提案のみ）

## 開発の基本姿勢

- 実装前にプロジェクトの `CLAUDE.md` / `AGENTS.md` / 設計書を読み、推測で実装しない
- コード変更後は、そのプロジェクトで定義された検証（lint・テスト・ビルド）を必ず実行する
- コミットは Conventional Commits 形式（英語）
